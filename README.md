# BubblesOS

![Image of the os running in Qemu](https://github.com/VegeSushi/BubblesOS/blob/main/md/Zrzut%20ekranu_20250926_194220.png?raw=true)

## Building

### Requirements

1. nasm
2. gcc-ia16 (can be installed from AUR)
3. make

### Actually building

```bash
make
```

## Running in Qemu

```bash
make run
```
